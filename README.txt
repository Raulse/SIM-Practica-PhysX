

En este ejercicio he partido de Snippet Articulation y he querido hacer un ejemplo de como afectan las diferentes masas a las colisiones entre objetos sólidos. Para ello:

-He creado un "vehiculo" que es una esfera que se mueve por teclado
-Con la tecla "espacio" podemos lanzar esferas con una masa muy pequeña

-He creado una rampa

Los controles son:

W, A, S, D: Para el control de la cámara
